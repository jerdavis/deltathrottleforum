Welcome to the forums!

Current design is three axis translation, so it does not replace rudders or stick twist.

There are possible delta geometries that allows for rotation as well, but I'm not sure if they would be feasible for a controller:
https://www.youtube.com/watch?v=qGyiXFGvkqE


Alternatively you could mount a handle to the handle base with one or more rotational axes, but again I'm unsure if it would be useful or there would be too much axis crosstalk.
https://www.youtube.com/watch?v=pcojMzFBodw

I myself am going to keep my rudders, since I don't have anything better to do with my feet and to avoid crosstalk as much as possible.
