To test it you can search for "joystick" or "USB Game Controller" in windows search and it will bring up the windows joystick configuration utility, then you can hit properties to view all of the axes.

In star citizen, you want to go to the controller settings, then go over to key bindings, then move the selector arrows over to joystick and press advanced controls customization. Since you now have two joysticks installed you are going to have to rebind both. It is pretty self explanatory, the only thing that could trip you up is for each strafe axis there is an option for "strafe left", "strafe right", and "strafe left/right". You want to pick "strafe left/right" and ignore the other two.

Let me know if that makes sense
